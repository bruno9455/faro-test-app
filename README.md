# Faro SDK Test Dashboard 📊 🧶

Este projeto é um ambiente de testes simples para o **Grafana Faro Web SDK**, configurado com **TypeScript**, **Vite** e gerido via **Yarn**.

O objetivo é validar o envio de telemetria (logs, erros e eventos) para uma instância do Grafana Cloud.
